## ?? Feature Gallery

### Core Features Showcase

**1. Multi-Provider File Scanning** ?
- Local file system (Windows, macOS, Linux)
- OneDrive (Microsoft Graph API)
- Extensible for future providers

**2. Intelligent Rules Engine** ??
- RegexPathRule (pattern matching)
- ExtensionRule (file types)
- AgeBasedRule (file age)
- CompositeRule (AND/OR logic)

**3. Naming Template System** ??
- 50+ built-in templates
- Variable substitution
- 12 custom functions
- Function chaining

**4. Advanced Duplicate Detection** ??
- Exact hash matching (SHA-256)
- Similarity detection
- Fuzzy matching
- Probabilistic approach

**5. High-Performance Database** ?
- SQLite with EF Core 9.0
- Sub-100ms queries
- 95%+ cache hit rate
- Optimized for 100k-1M files

**6. OneDrive Integration** ??
- Microsoft Graph API
- OAuth 2.0 authentication
- Chunked uploads (320KB)
- Progress tracking
